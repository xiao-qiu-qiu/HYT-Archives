## 生存玩法存档说明

保存内容：

- 只保存主世界
- 保存从出生点到边界内的方块内容
- 尽量保存整个可进入世界范围

未能完整保存或存在问题的内容：

- 不保存箱子内容物
- 1.12.2 的存档无法直接显示展示框内的地图画（实体缺失），但地图数据可能存在于存档 `data` 文件夹中
- 存档内箱子模型可能不会立即渲染，打开一次后通常会恢复
- 非 1.21.4 的存档都保存了告示牌，用 1.21.4 存的存档未能保存
- 1.21.4 的存档可以保存实体数据，1.12.2 的存档无法保存，展示框等实体内容缺失

以上问题与花雨庭服务端发包方式、存图模组 bug 等因素有关。已经尝试修改存图模组并试过很多办法，但时间紧迫，本人能力有限，只能尽量保留。
